dom语法

在jQuery中我们可以通过选择器快速获取元素,这些新的选择器,在html5中提供了一个类似的方法

querySelector方法

返回当前文档中第一个匹配特定选择器的元素

var node = document.querySelector(css选择器);

querySelectorAll

返回当前文档中所有匹配特定选择器的元素（返回值是数组）

var nodeLists = document.querySelectorAll(css选择器);

classList

这个元素返回的是dom元素的class列表

// 获取元素
        var dom = document.querySelector("header");

add:添加一个class到元素的class列表中 dom.classList.add("top");

remove:从元素的class列表中移除一个class dom.classList.remove('pink');

toggle:切换一个class是否存在于一个元素的class列表中 有就删除 没有就添加 dom.classList.toggle('pink');

contains:查询一个指定的class是否存在于class列表中 alert(dom.classList.contains('orange'));

h5data*属性

data-:这个是固定格式

date-xxx:xxx是自定义的值

属性名不应该有大写 并且在data-后至少得有一个字符

属性值可以是任意字符串

不要使用纯数字 特殊字符等作为自定义值

如果data之后使用-连接多个值，取值时 需要驼峰命名法
